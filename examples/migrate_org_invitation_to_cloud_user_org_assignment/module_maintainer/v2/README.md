# Module Example: Cloud User Org Assignment (with team assignments)

This module shows the migrated implementation using `mongodbatlas_cloud_user_org_assignment` (and optional `mongodbatlas_cloud_user_team_assignment` for team memberships). A `moved` block maps from the legacy `mongodbatlas_org_invitation` resource.

If you are migrating from `mongodbatlas_org_invitation`, see the migration guide for import ID formats and state-move guidance:
- `docs/guides/atlas-user-management.md`
