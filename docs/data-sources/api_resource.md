# Data Source: mongodbatlas_api_resource

`mongodbatlas_api_resource` reads any Atlas Admin API `GET` endpoint and exposes the full response through the `output` Dynamic attribute. Useful for endpoints that do not yet have a typed data source, or for preview features.

~> **IMPORTANT — `output` is NOT marked Sensitive.** Any secret returned by the API surfaces in `terraform plan` / `terraform apply` / `terraform show`. Pipe sensitive fields through a Terraform `output { sensitive = true }` block when consuming them.

## Example Usage

```terraform
# Minimal example: manage an Atlas Service Account via the generic resource.
# A typed `mongodbatlas_service_account` resource exists — prefer it for production.

resource "mongodbatlas_api_resource" "this" {
  path                  = "/api/atlas/v2/orgs/${var.org_id}/serviceAccounts"
  id_attribute          = ["clientId"]
  create_only_body_keys = ["secretExpiresAfterHours"]

  body = {
    name                    = "example-generic-sa"
    description             = "Service account managed via mongodbatlas_api_resource"
    roles                   = ["ORG_READ_ONLY"]
    secretExpiresAfterHours = 24
  }
}

data "mongodbatlas_api_resource" "this" {
  path = mongodbatlas_api_resource.this.id
}

output "service_account_client_id" {
  value = mongodbatlas_api_resource.this.output.clientId
}

output "service_account_first_secret" {
  description = "Returned only on Create. After the first refresh this is null."
  value       = try(mongodbatlas_api_resource.this.output.secrets[0].secret, null)
  sensitive   = true
}
```

## Schema

### Required

- `path` (String) Atlas API path to GET.

### Optional

- `preview` (Boolean) Shorthand for `version_header = "application/vnd.atlas.preview+json"`. Mutually exclusive with `version_header`.
- `version_header` (String) Atlas API version media type. When unset, defaults to today's UTC date in the form `application/vnd.atlas.<YYYY-MM-DD>+json` (Atlas snaps the date down to the latest published version on or before it). Mutually exclusive with `preview`.

### Read-Only

- `id` (String) Synthetic identifier (the path).
- `output` (Dynamic) Full API response from the most recent successful operation.

See the [`mongodbatlas_api_resource` resource documentation](../resources/api_resource.md) for the full description of `version_header`, supported endpoints, and the `output` sensitivity warning.
